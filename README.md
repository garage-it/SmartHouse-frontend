# How to use
- install node dependencies
```sh
  npm install
```

- run application
```sh
  npm start
```

If you need to set up environment variables - just copy ```.env.example``` to ```.env``` and feel free to set up

- run unit tests
```sh
  npm test
```

- run eslint validation
```sh
  npm run validate
```

- run eslint validation & automatically fix problems
```sh
  npm run lintfix
```

- build application
```sh
  npm run build
```
